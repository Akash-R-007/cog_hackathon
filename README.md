# Fraud Detection Dashboard

A full-stack dashboard for bank transaction monitoring and fraud detection.

## Backend
- Node.js + Express + MongoDB
- REST API for account transactions

## Frontend
- React (Vite) + Tailwind CSS
- Dashboard, Transactions, Reports, Settings, Profile

## Getting Started

### Backend
1. `cd backend`
2. `npm install`
3. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
